# Family Health WhatsApp Tracker

This is a personal, non-commercial project to help elderly parents
track medication adherence and health test results using WhatsApp only.
It sends reminders, logs responses, allows test result uploads, and shows simple trends and correlations over time.
It does not give medical advice. It helps families and doctors have better conversations.

## Goals
- Simple medication reminders
- Easy logging via WhatsApp
- Upload test results (text or photo)
- View trends and correlations over time

## Non-Goals
- No medical advice
- No diagnosis
- No treatment recommendations
